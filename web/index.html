Try It Now

Get a scoped token and query OpenSearch in 30 seconds.

1. Start the proxy

docker compose up

2. Get a scoped token

curl -X POST http://localhost:8443/oauth/token \
  -d "grant_type=client_credentials" \
  -d "client_id=my-agent" \
  -d "client_secret=secret" \
  -d "scope=read:logs-*"

3. Query with the token

curl -H "Authorization: Bearer <token>" \
  http://localhost:8443/logs-*/_search \
  -d '{"query":{"match":{"level":"error"}}}'

4. Revoke when done

curl -X POST http://localhost:8443/oauth/revoke \
  -d "token=<token>"

Features

🔑

JWT Validation

JWKS auto-discovery, RS256/ES256. Validates tokens from any OIDC provider.

🎯

Scope-to-Role Mapping

read:logs-* → OpenSearch logs_reader role. Configurable in YAML.

🔄

Unified Auth

Single entry point for Engine + Dashboards APIs. Zero changes to existing components.

📋

Token Lifecycle

Issue, revoke, list, inspect tokens via REST API. Full audit trail.

🌲

Cedar Policies

Fine-grained access control with Cedar policy language. Evaluate locally, no network calls.

⚡

CLI Tool

oauth4os login, create-token, revoke, status. Developer-friendly.
