Features

🔑

JWT Validation

JWKS auto-discovery, RS256/ES256. Validates tokens from any OIDC provider.

🎯

Scope → Role Mapping

read:logs-* → OpenSearch logs_reader role. Configurable in YAML.

🔄

Unified Auth

Single entry point for Engine + Dashboards APIs. Zero changes to existing components.

📋

Token Lifecycle

Issue, revoke, list, inspect tokens via REST API. Full audit trail.

🌲

Cedar Policies

Fine-grained access control with Cedar policy language. Evaluate locally, no network calls.

⚡

CLI Tool

oauth4os login, create-token, revoke, status. Developer-friendly.
