2. Docker 30 seconds

⏱ One command — needs Docker installed

Run the proxy locally with built-in demo data. No OpenSearch needed.

  1. Start the proxy: 
    docker run -p 8443:8443 jianghuan/oauth4os:latest
    Expected: "oauth4os listening on :8443"
  2. Open in your browser: 
    http://localhost:8443
    Expected: oauth4os landing page with links to demo, playground, admin
  3. Get a token (optional — test the API): 
    curl -s -X POST http://localhost:8443/oauth/token \
  -d 'grant_type=client_credentials&client_id=demo-agent&client_secret=demo-secret&scope=read:logs-*' | python3 -m json.tool
    Expected: {"access_token": "eyJ...", "token_type": "Bearer", "expires_in": 3600}

3. Full Stack 2 minutes

⏱ Proxy + OpenSearch + Dashboards — needs Docker Compose

Run the complete stack: oauth4os proxy, OpenSearch engine, and OpenSearch Dashboards.

  1. Clone and start: 
    git clone https://github.com/seraphjiang/oauth4os.git
cd oauth4os
docker compose up
    Expected: 3 containers start — oauth-proxy, opensearch, dashboards. Takes ~60s on first run.
  2. Wait for "oauth4os listening on :8443", then open: 
    http://localhost:8443          # oauth4os proxy + demo
http://localhost:5601           # OpenSearch Dashboards (direct)
  3. Search through the proxy: 
    TOKEN=$(curl -s -X POST http://localhost:8443/oauth/token \
  -d 'grant_type=client_credentials&client_id=demo-agent&client_secret=demo-secret&scope=read:logs-*' | python3 -c "import sys,json;print(json.load(sys.stdin)['access_token'])")

curl -s http://localhost:8443/logs-*/_search \
  -H "Authorization: Bearer $TOKEN" \
  -d '{"query":{"match_all":{}},"size":5}' | python3 -m json.tool
    Expected: JSON with hits from OpenSearch, proxied through oauth4os with JWT auth

⚠️ Troubleshooting

Docker build fails?

# Pull the pre-built image instead of building:
docker pull jianghuan/oauth4os:latest
docker run -p 8443:8443 jianghuan/oauth4os:latest

OpenSearch won't start (Mac/ARM)?

# Increase Docker memory to 4GB:
# Docker Desktop → Settings → Resources → Memory → 4GB
# Then retry:
docker compose down && docker compose up

Port 8443 already in use?

# Use a different port:
docker run -p 9443:8443 jianghuan/oauth4os:latest
# Then open http://localhost:9443

"Cannot connect to Docker daemon"?

Start Docker Desktop first, wait for the whale icon to stop animating, then retry.
