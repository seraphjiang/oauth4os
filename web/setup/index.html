Get oauth4os running in 3 steps. ← Back

Start the proxy

Run with Docker (no install needed):

docker run -p 8443:8443 jianghuan/oauth4os:latest

Get a token

Use client credentials to get an access token:

curl -X POST http://localhost:8443/oauth/token \
  -d 'grant_type=client_credentials&client_id=demo-agent&client_secret=demo-secret&scope=read:logs-*'

Query OpenSearch

Use the token to search through the proxy:

curl http://localhost:8443/logs-*/_search \
  -H 'Authorization: Bearer <token>' \
  -d '{"query":{"match_all":{}},"size":5}'
